HeroiCommits

Show your commitment with commits

HeroiCommits lets you graph the commits you made in a repo. An HeroiCommit is a 
commit made outside the standard 9-18 hour or in a weekend.

1. Open a folder with a git repo. Make sure that you are in the branch you want to check the commits and you made a pull.

2. Press the •HeroiCommits• button on the status bar at the bottom right side.

3. Select a year.

4. Select a month.

5. Press the "Search HeroiCommits" button.

6. The HeroiCommits and regular commits are shown in the graph
